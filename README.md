## Project Overview

This project implements a complete enterprise data warehouse on Microsoft Azure following Bill Inmon's classical data warehouse methodology, incorporating Change Data Capture (CDC) for near-real-time data ingestion.

### Key Achievements
- **Latency Reduction:** 24 hours → 8-12 minutes (144x improvement)
- **Query Performance:** 6+ minutes → <3 seconds (120x improvement)
- **Storage Efficiency:** 75% reduction using Parquet format
- **Data Quality:** 99.95% completeness, 100% consistency
